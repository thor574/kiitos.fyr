# Test Addy Osmani sitt agent-skills-repo

**Status:** åpen  
**Prioritet:** lav  
**Estimat:** kort

## Bakgrunn

Addy Osmani, direktør for Cloud AI i Google, har laget repoet [agent-skills](https://github.com/addyosmani/agent-skills). Repoet bør testes som mulig inspirasjon for kiitos, særlig fordi kiitos allerede bruker skills og agentinstrukser som sentrale byggesteiner.

## Oppgave

Test repoet lokalt eller i en avgrenset sandkasse og vurder om det finnes konkrete mønstre, arbeidsmåter eller ferdige skills som det er verdt å tipse andre om å prøve ut.

## Vurderingsspørsmål

1. Hvilke skills eller arbeidsflyter i repoet er umiddelbart nyttige for utviklere som bruker KI-agenter?
2. Finnes det ideer som bør sammenlignes med eller speiles i `kiitos.fyr/.github/skills/`?
3. Er repoet lett å ta i bruk uten mye lokal oppsettstid?
4. Er det noe vi bør anbefale som et lavterskel eksperiment for andre?

## Foreslått leveranse

- Kort notat med hva som ble testet.
- Én anbefaling: tips videre, følg med, eller la ligge.
- Eventuelle konkrete forslag til kiitos-oppgaver hvis repoet viser nyttige mønstre.

## Risiko ved å vente

Lav. Dette er et utforskingsspor, men det kan være nyttig å fange opp gode mønstre tidlig mens repoet er ferskt og relevant.
