# Evaluerings-sjekkliste

Sjekkliste for å validere at evalueringer oppfyller Bastøes fem forutsetninger og kvalitetsstandarder.

## Hurtigoppslag

- **Formål**: Verifisere fullstendighet og kvalitet under evalueringsprosessen
- **Brukes av**: Evalueringsleder, peer-reviewer, kvalitetssikrer
- **Når**: Under evaluering, før validering, før lukking

---

## Forutsetning 1: Klargjøring av formål

- [ ] `evaluering-formaal.md` eksisterer
- [ ] Formål er tydelig definert (hva evalueringen skal brukes til)
- [ ] Evalueringstype er spesifisert (prosess/resultat/effekt/lærdom)
- [ ] Beslutningsstøtte-formål er angitt
- [ ] Læringsformål er angitt (hvis relevant)
- [ ] Ansvarliggjøring er angitt (hvis relevant)
- [ ] Evalueringsspørsmål er konkrete og målbare
- [ ] Formål er godkjent av prosjekteier

---

## Forutsetning 2: Avklaring av omfang og metode

- [ ] `evaluering-omfang.md` eksisterer
- [ ] Evalueringsomfang er avgrenset (hva som inkluderes/ekskluderes)
- [ ] Metode er valgt og begrunnet
- [ ] Metode er egnet for evalueringsspørsmålene
- [ ] Datakilder er identifisert
- [ ] Datainnsamlingsmetode er beskrevet
- [ ] Oppfølgingsprosess er definert
- [ ] Tidsplan er realistisk
- [ ] Ressursbehov er estimert

---
## Forutsetning 3: Systematiske prosesser

- [ ] Arbeidsflyt følger [evalueringsflyt.md](evalueringsflyt.md)
- [ ] Fase 1 (Initiell) er fullført
- [ ] Fase 2 (Datainnsamling) er fullført
- [ ] Fase 3 (Analyse) er fullført
- [ ] Fase 4 (Rapportering) er fullført
- [ ] Alle faser er dokumentert i journal
- [ ] OECD DAC-kriterier er vurdert
- [ ] Norad-standarder er fulgt
- [ ] Prosessen er gjentakbar

---
## Forutsetning 4: Kvalitetsstandard

- [ ] Kvalitetskriterier fra [kvalitetsstandard-evaluering.md](../styringsgrunnlag/kvalitetsstandard-evaluering.md) er vurdert
- [ ] Relevans: Evalueringen adresserer informasjonsbehov
- [ ] Nøyaktighet: Data og funn er korrekte
- [ ] Pålitelighet: Metode og prosess er troverdige
- [ ] Nyttighet: Resultater er praktisk brukbare
- [ ] Effektivitet: Ressursbruk er proporsjonal
- [ ] Etikk: Evalueringen er etisk forsvarlig
- [ ] `/maskin evaluering-kvalitet` er kjørt
- [ ] `kvalitetsrapport.md` eksisterer

---
## Forutsetning 5: Organisasjons- og læringskultur

- [ ] Handlingsplan er utarbeidet
- [ ] Anbefalinger er prioriterte
- [ ] Ansvarlige er tildelt for hver anbefaling
- [ ] Lærdommer er dokumentert
- [ ] Lærdommer er delt i `guide/lærdommer/`
- [ ] Resultater er kommunisert til interessenter
- [ ] Evalueringen er arkivert
