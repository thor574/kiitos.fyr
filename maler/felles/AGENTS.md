# AGENTS.md (laugrot)

Denne fila gjelder for hele katalogtreet under denne rota.

For operative instrukser i dette lauget:
1. Les `./velkommen.md`.
2. Følg instruksjonskjeden derfra.

For universell kiitos-orientering, se fyret.

## Tilgjengelige agenter

Agentene ligger lokalt i `.github/agents/` i denne laugroten. De er generert fra malene i `kiitos.fyr/maler/felles/.github/agents/` og tilpasset dette laugets prosjekter.

| Agent | Rolle |
|---|---|
| **agent001** | Arbeidsrom-speider |
| **agent002** | Instruksflate-vokter |
| **agent003** | Oppgave-triager |
| **agent004** | Review-vakt |
| **agent005** | Harmoniseringsspeider |

Alle agenter er lesende spesialister — ingen av dem gjør kodeendringer.
